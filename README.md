# NASA Space Explorer Gallery

Responsive starter site for exploring NASA Astronomy Picture of the Day (APOD) content by date range.

## Features

- Responsive layout for mobile and desktop.
- Top title bar with NASA logo and page title: Space Explorer.
- Date range search (start date + end date).
- Live fetch from the NASA APOD API.
- Image cards with title, date, and short description.

## Files

- `index.html` - page structure and controls.
- `styles.css` - responsive styling and visual design.
- `script.js` - API calls, validation, and gallery rendering.
- `nasa-logo.svg` - logo used in the title bar.

## Set Your API Key

Open `script.js` and replace:

`const API_KEY = "YOUR_NASA_API_KEY";`

with your NASA API key.

## Run Locally

Open `index.html` in your browser, or serve the folder with any static server.
